## Build / Development

1. Clone project and install dependencies

```sh
git clone https://github.com/MarioMatheus/snowboard-emojedition.git
cd snowboard-emojedition
npm install
```

2. Build and run project

```sh
spago build
npm run dev
```

3. Go to http://localhost:1234
